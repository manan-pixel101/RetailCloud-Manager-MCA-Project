# RetailCloud Manager

Implementation of Cloud Computing in Small Business Operations using AWS Services: A Practical Approach

## Technologies Used
- Amazon S3
- AWS Lambda
- API Gateway
- DynamoDB
- IAM
- CloudWatch
- CloudTrail
- HTML
- JavaScript

## Features
- Secure business document upload
- Approval workflow
- Role-based access
- Monitoring and auditing
- Cloud-based storage management

## Architecture
Frontend → API Gateway → Lambda → Amazon S3 + DynamoDB
